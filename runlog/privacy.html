プライバシーポリシー

本ポリシーは、iOSアプリ「runlog」(以下「本アプリ」)における利用者情報の取り扱いについて定めるものです。

1. 位置情報(GPS)

本アプリは、ランニングの距離・ペース・ルートを計測するために位置情報を使用します（画面ロック中の継続計測を含む）。計測データは原則として利用者の端末内に保存されます。

2. アカウントとクラウド同期

クラウド同期機能を利用する場合のみ、メールアドレスとパスワードによるアカウントを作成します。アカウント情報とランの記録は、クラウドサービス Supabase 上に保存され、行レベルセキュリティにより本人のみがアクセスできます。クラウド同期を利用しない場合、データは端末内のみに保存されます。

3. ランの記録

距離・時間・ペース・ルート座標・獲得標高・メモ等の記録は、端末内に保存され、ログイン時のみクラウドに同期されます。

4. AIコーチ機能について

AIコーチ機能は任意です。利用者がご自身の Google Gemini API キーを設定した場合に限り、直近の練習データの要約が助言生成のため Google の Gemini API に送信されます。APIキーは利用者の端末内(SecureStore)にのみ安全に保存され、開発者が取得することはありません。

5. 外部サービス

本アプリは、地図表示にAppleのマップ、クラウド同期にSupabase、AIコーチ(任意)にGoogle Geminiを利用します。各サービスの取り扱いは各社のポリシーに従います。

6. 広告・トラッキング

本アプリは広告を表示せず、広告識別子(IDFA)の取得やユーザートラッキングを行いません。

7. お問い合わせ

本ポリシーに関するお問い合わせは サポートページ よりお願いいたします。

8. 改定

本ポリシーは、法令の変更や機能追加に応じて改定されることがあります。
